Hello from AWS CloudFront and S3!

This is a sample static website hosted on Amazon S3 and delivered through Amazon CloudFront.

Features of this setup:

  • Private S3 bucket with static website hosting enabled
  • CloudFront distribution for secure content delivery
  • Infrastructure as Code using Terraform
  • Automated deployment through GitHub Actions
